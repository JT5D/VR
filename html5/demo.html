Virtual Reality in WebGL Demo

This WebGL application is to demonstrate the use of open source software and open standards to be able to create 3D scenes that can be viewed in a progressively enhanced way. The application is capable of rendering as a regular 3D graphics application, or it can operate in anaglyph red/cyan mode and stereo viewer mode.
